# VoxelCivitas

**A virtual civilization lab for agent-based research.**

VoxelCivitas is a web-based platform for designing, running, and analyzing controlled multi-agent experiments inside Minecraft-compatible virtual worlds. Researchers can configure agents, launch simulations, collect behavioral metrics, visualize emergent patterns, compare experimental runs, and generate reproducible research reports.

Created by **Mark Jacinto Kropf** as an independent research engineering project.
